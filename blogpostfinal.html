Blog Post Final - 12/7/21

In my personal opinion, I would focus on building one iterative project for this class.
Where students would select a main project idea that would incorporate all the major elements of the course design.
This would make it so that the class would be iterative and the project would be similar to project lifecycles in industry.
The mini-labs and the labs were useful and extremely effective for learning, they are just very demanding.


What would be helpful to my students is the book. The book is amazing resource that was extremely useful to me.
Other resources I used are YouTube and Stack Overflow, which will also be very useful.
What could have been useful is the access to a TA that has experience in Web Development.
Overall, the course is designed very well and can be improved by having TAs to help with extra questions.
