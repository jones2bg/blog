This week I learned about a fullstack development package-based utility called Node.js.
Node.js is very useful and powerful, providing a package manager, npm, to quickly install and use packages.
Much like how pip works, you would just install any given Node.js package and off you go!
Node.js is built off of JavaScript which is both a front end and backend tool.

The following instructions are how you install Node.js on Ubuntu systems:
sudo apt update
sudo apt install nodejs
sudo apt install npm
Personally, I have only scraped the surface of the power of Node.js.
I have only ever used Node.js with Vue.js and Vuetify with Vuex, a Node.js package.
To understand the depth of Node.js, I looked into React.js, here , and Angular.js, here.
To my surprise, they are very similar to Vue.js, and I am wondering now, why was I so hesitant to start coding in either of these languages.
Overall, my studies yielded an important lesson: Don't judge a book by it's cover.
