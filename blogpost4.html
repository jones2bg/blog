Blog Post 4 - 9/28/21

This week I decided to explore a cool JavaScript library called React.js.
React is a really, really versatile library and can provide solutions for a broad range of problems.
What makes this library advantageous over angularjs and Vue.js, you might ask?
Well, it's pretty clear... the previous mentioned libraries and frameworks are built to specifications for applications specific to that developer.
"But, what does that mean?" - Essentially, each library and framework are built to be different from the other.
There is no true use for each of the aforementioned libraries and frameworks. Just use what is your favorite!

Anyways... I got off topic accidentally.

So, React has tons of cool things you can explore and try out.
The coolest thing that React has, is the RenderDOM. Which is essentially an element/constant renderer
This renderer allows you to display JavaScript content on the page with some some extra spice.
In conclusion, React is awesome and allows you to do a lot of fun things with it.
